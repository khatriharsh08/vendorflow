3. TESTING WORKFLOWS
-----------------------------------------

[WORKFLOW 1: VENDOR REGISTRATION]
1. Go to Login page -> Click "Sign Up".
2. Register as a new user (e.g., vendor@test.com / password).
3. You will be redirected to the Vendor Onboarding form.
4. Fill in the details using "SCENARIO A" data above.
5. Submit the form.
6. Status should be: "Under Review".

[WORKFLOW 2: VENDOR APPROVAL (OPS TEAM)]
1. Logout of Vendor account.
2. Login as "ops@vendorflow.com" (Password: password).
3. Go to "Vendors" -> "Pending Approvals".
4. You should see "TechFlow Solutions".
5. Click "View" -> "Verify Documents" -> "Approve Vendor".
6. Vendor status changes to: "Active".

[WORKFLOW 3: PAYMENT REQUEST]
1. Login as the approved Vendor (vendor@test.com).
2. Dashboard should now show full access.
3. Go to "Payments" -> "Request Payment".
4. Enter Amount: 5000, Description: "Service Fee".
5. Submit.

[WORKFLOW 4: PAYMENT APPROVAL (FINANCE TEAM)]
1. Login as "finance@vendorflow.com".
2. Go to "Payments".
3. You should see the request for 5000.
4. Click "Approve".
